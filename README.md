## Site local pret pour Vercel

Ce projet utilise [Next.js](https://nextjs.org) et est deja configure pour:

- developpement local rapide
- build de production
- deploiement simple sur Vercel

## Demarrage local

```bash
npm install
npm run dev
```

Puis ouvre [http://localhost:3000](http://localhost:3000).

## Commandes utiles

- `npm run dev` : serveur de developpement
- `npm run build` : build production
- `npm run start` : lancer la version production localement
- `npm run lint` : verifier le code

## Deploiement Vercel

1. Cree un repo GitHub avec ce dossier.
2. Connecte le repo a [Vercel](https://vercel.com/new).
3. Garde les parametres par defaut (`Framework Preset: Next.js`).
4. Clique sur **Deploy**.

Le projet est pret. La prochaine etape est de definir le nom du site et son identite visuelle.
